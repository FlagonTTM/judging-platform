Оценка хакатона в один клик — judging platform

───────────────────────────────────────────────────────────

ПРОБЛЕМА

Организаторам хакатона приходится вручную сводить анкеты, таблицы
оценок и комментарии от нескольких жюри. Это занимает часы, провоцирует
ошибки в подсчётах и создаёт непрозрачные итоги для команд.

РЕШЕНИЕ

Веб-приложение, которое закрывает весь путь хакатона — от регистрации
команд до публичного рейтинга — в одной системе с чистым scoring,
понятным жюри и прозрачным для участников.

───────────────────────────────────────────────────────────

ЧТО ВНУТРИ

• Роли admin / judge / team с JWT-авторизацией и защищёнными роутами
• Конструктор критериев с весами 0–100 и контролем суммы = 100
• Панель жюри: ползунки, комментарии, черновики и финальная отправка
• Автоматический weighted scoring и агрегация по нескольким жюри
• Live-leaderboard с выделенным лидером и обновлением каждые 10 секунд
• Прогресс-матрица «команда × этап» для организатора
• Импорт команд из Google Sheets по одной ссылке
• Team Preview «как жюри увидит ваш проект» — эвристический анализ
  описания: one-liner, фичи, покрытие критериев, слабые места,
  вероятные вопросы жюри
• Submission Readiness Check — проверка, что в заявке есть демо, репо,
  описание, видео и скриншот; HEAD-проверка живости ссылок

ФОРМУЛА ПОДСЧЁТА

judgeScore = Σ( value / maxScore * weight )   // в шкале 0 – 100
finalScore = avg( judgeScore по всем жюри )
ranking   = sort(finalScore DESC)

───────────────────────────────────────────────────────────

ТЕХНОЛОГИИ

Frontend  React 18 + TypeScript + Vite + Tailwind + React Query
Backend   FastAPI + SQLAlchemy + Pydantic v2 + Alembic
DB        PostgreSQL
Auth      JWT в httpOnly-куках
Deploy    Docker Compose
Tests     vitest (frontend) + pytest (backend)

АРХИТЕКТУРА

Сознательно выбран монолит: одна бизнес-логика, один деплой, одна
история миграций. Scoring считается на сервере, фронт только рисует.
Каждая сущность (Event, Team, Criterion, Score, Stage, TeamProgress,
Submission) имеет собственный REST-контракт и Alembic-миграцию.
